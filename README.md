# Handwritten Digit Recognition Using CNN in MATLAB

## Project Overview
This project implements a Convolutional Neural Network (CNN) for handwritten digit recognition using MATLAB. The model is trained on MATLAB’s built-in handwritten digit dataset and is capable of classifying digits from 0 to 9 with high accuracy.

## Features
- Handwritten digit classification
- CNN-based deep learning model
- Data augmentation for improved generalization
- Confusion matrix visualization
- Sample prediction visualization

## Tools and Technologies
- MATLAB
- Deep Learning Toolbox
- CNN (Convolutional Neural Network)

## Results
The trained model achieved high classification accuracy on the testing dataset and successfully recognized most handwritten digits.
